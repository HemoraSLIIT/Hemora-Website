Deep Learning + Microscopic Blood Smear Analysis

Hemora

Identification of Hematological Diseases using Deep Learning and Microscopic Blood Smear Image Processing

Hemora is a public-facing final-year research project focused on automated identification of hematological diseases through blood smear images and Complete Blood Count data. The platform is designed to support faster, more reliable screening in Sri Lankan clinical environments where specialist access may be limited.

Project Scope

Comprehensive overview of our research approach

Literature Survey

Hematological diseases remain a major clinical concern, and diagnosis often depends on manual microscopy, CBC interpretation, and expert review. This process can be slow, observer-dependent, and difficult to scale. Recent deep learning methods have shown strong potential in medical image classification, and combining visual smear analysis with CBC parameters can improve decision quality.

Research Gap

  • Current workflows are frequently manual and time-consuming.
  • Specialist access is limited in low-resource settings.
  • Many solutions rely on only image data or only CBC data.
  • Sri Lanka-focused hybrid detection systems are limited.
  • Explainable web-ready clinical tools are often missing.

Research Problem

Early and accurate diagnosis of hematological diseases remains difficult in low-resource settings due to manual processes, diagnostic delays, limited specialist availability, and lack of scalable automated systems that combine blood smear image analysis with CBC data.

Main Objective

Develop an automated system for identifying hematological diseases using microscopic blood smear image processing and deep learning, with focus on ALL, Beta-Thalassemia Minor, IDA, and SCD.

Sub Objectives

  • Detect disease-specific morphological cell patterns.
  • Integrate CBC parameters into hybrid predictions.
  • Evaluate models using local clinical datasets.
  • Deliver a user-friendly web interface for professionals.

Methodology

The methodology includes CBC analysis, image analysis, hybrid decision-making, and result generation.

01

User Access and Roles

Authenticated access is handled through the React frontend and JWT-protected backend APIs for Admin, Doctor and Lab Technician roles.

02

Patient Registration

The platform collects patient details, one or more blood smear images, and a CBC report file through a structured web form.

03

CBC Extraction

CBC reports are processed using pdfplumber first, with OCR fallback through PyMuPDF and pytesseract for scanned files.

04

CBC-Based Analysis

Parsed parameters are validated and scored against disease-specific rules to compute suspicion levels and matched clinical criteria.

05

Disease-Wise Models

ALL - YOLOv8s + EfficientNetB0
Beta-Thal (Minor) - YOLOv8s + ResNet50
SCD - YOLOv8n + R-CNN
IDA - YOLOv8s + ResNet50

06

Image Analysis

During analysis, the system identifies and marks suspicious cells, evaluates the overall blood smear image.

07

Hybrid Decision Layer

Final disease scoring combines 40% CBC analysis and 60% image analysis, with CBC-only fallback when image models are unavailable.

08

Review and Reporting

Results are stored, reviewed, re-generated when needed, and presented with annotated images, doctor feedback, and PDF report export support.

09

System Integration

The complete workflow is deployed as a full-stack platform using Docker Compose, PostgreSQL, Django services, and the React frontend.

Disease Focus

Four Targeted Hematological Conditions

Each disease pathway combines visual morphology cues with CBC indicators to support accurate, practical screening.

ALL

Acute Lymphoblastic Leukemia

Focuses on lymphoblast detection from smear images together with parameters such as WBC count and hemoglobin.

BTM

Beta-Thalassemia Minor

Focuses on microcytosis, hypochromia, and target cells using CBC cues like MCV, MCH, and RBC count.

SCD

Sickle Cell Disease

Focuses on sickle-shaped RBC identification with supporting CBC features to reduce manual burden.

IDA

Iron Deficiency Anemia

Focuses on microcytosis, hypochromia, anisocytosis, and poikilocytosis using hemoglobin, MCV, and MCH.
